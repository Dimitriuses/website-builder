# Images Folder

Place your website images here.

Images will be automatically copied to the build folder when you run `node build.js`.

## Usage

Reference images in your components or content:
```html
<img src="assets/images/logo.png" alt="Logo">
```

## Supported Formats
- .jpg / .jpeg
- .png
- .gif
- .svg
- .webp

## Organization Tips
- Keep images organized in subfolders (e.g., `images/gallery/`, `images/team/`)
- Use descriptive filenames
- Optimize images before adding (use tools like TinyPNG)
